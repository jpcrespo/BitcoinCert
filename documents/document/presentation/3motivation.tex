\begin{frame}{Motivation}
\framesubtitle{Trade-offs in Turing-Complete Networks}

Other blockchain networks (e.g., Ethereum) adopt an account-based model to support expressive logic and smart contracts.\\
\vspace*{0.2cm}
While this programmability enables sophisticated applications, it introduces several trade-offs:

\begin{itemize}
    \item \textbf{Increased attack surface:} Greater implementation complexity expands the potential for vulnerabilities.
    \item \textbf{Consensus and execution overhead:} On-chain computation increases resource requirements and operational costs.
    \item \textbf{System fragility:} Complex execution environments are more susceptible to denial-of-service vectors and unintended behaviors.
\end{itemize}
\end{frame}
